# Project Overview

This project is a simple REST API for managing a library system that includes functionalities for managing books and authors. It employs a modern tech stack to ensure performance, security, and ease of development.

## Tech Stack Details
- **Framework:** Spring Boot
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Build Tool:** Maven

## Architecture Decisions
- Adopted the Spring Boot framework for rapid development and ease of integration with other tools.
- Utilized PostgreSQL for its reliability and advanced features suitable for handling complex queries.
- Implemented JWT for secure and stateless authentication, ensuring scalability of the application.
