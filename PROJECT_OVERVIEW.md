# Project Overview

The Library API is a simple RESTful service designed for managing a library system of books and authors.

## Tech Stack
- **Java 17**
- **Spring Boot 3.x**
- **PostgreSQL**
- **JWT for Authentication**
- **Maven** for Build Management

## Architecture Decisions
- **RESTful API Design**: Follows REST principles for clear and efficient interaction.
- **Spring Data JPA**: Simplifies database interactions with PostgreSQL.
- **Security**: Utilizes JWT for stateless authentication and Role-Based Access Control (RBAC).
